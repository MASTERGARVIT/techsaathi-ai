TechSaathi is an AI-powered learning and productivity assistant designed for Bharat.
It simplifies complex technical concepts, codebases, and workflows using contextual,
step-by-step explanations tailored to user skill levels.

## Problem
Technology learning in Bharat is slowed by complex, English-centric documentation and
lack of practical, guided explanations.

## Solution
TechSaathi acts as a digital mentor that explains, guides, and helps users take action
through AI-powered contextual understanding.

## Key Features
- Role-based onboarding (Student / Developer / Non-technical)
- Simplified explanations using Indian context
- Code and document analysis
- Step-by-step actionable guidance
